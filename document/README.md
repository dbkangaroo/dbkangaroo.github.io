## Dependencies
Kangaroo build on GLib / GTK3 / Vala, so we must install them first, here's the list of dependencies required:
 - `gtk+-3.0 >= 3.18`
 - `gdk-pixbuf-2.0`
 - `glib-2.0`
 - `gee-0.8`
 - `gobject-2.0`
 - `gtksourceview-4.0`
 - `libjson-glib-1.0`
 - `libsoup-2.4`
 - `libgda-5.0`
